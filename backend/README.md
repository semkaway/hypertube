# hypertube_back

> Hypertube Node.js

## Build Setup

__First install `MongoDB` and `Docker` from `MSC`__ 

``` bash
# install dependencies
npm install

# start database in seperate terminal window
npm run db

# run Docker desktop app
# wait till it have status `running`

# then start node api
npm start
```
